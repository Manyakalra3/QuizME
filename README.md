# QuizME
## QuizME is a test engine app.
##Features
* Test Group creation
* Test Group deletion
* Question addition
* Question deletion
* Assigning roles(for teachers and students separately)
* Authentication
* neat and clean UI

## TECH STACK USED
### FRONTEND
* React
* React Routing
* Material icons
### Backend
* Node js
* ExpressJS
* Firebase
### DATABASE
* MongoDB
